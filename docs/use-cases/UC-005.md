# UC-5: Delete Game

**Actor:** Author

**Preconditions:**
- The game exists in the store

**Main Flow:**
1. Author supplies a game-id
2. System displays the game metadata and asks for confirmation
3. Author confirms
4. System deletes the game directory and all files within it

**Alternate Flows:**
- **game-id not found** — system reports the error; nothing is deleted
- **author cancels** — system leaves the game unchanged

**Postconditions:**
- The game no longer exists in the store
