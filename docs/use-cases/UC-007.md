# UC-7: Remove Turning Point

**Actor:** Author

**Preconditions:**
- A session is open for the game
- The turning point to remove exists and is not ply 1

**Main Flow:**
1. Author supplies a turning point ply
2. System validates that the ply is an existing turning point and is not the first turning point
3. System removes the `[%tp]` marker at that ply from the working annotated PGN
4. System removes the corresponding segment entry from the working annotation JSON
5. System derives the updated segment list and returns it

**Alternate Flows:**
- **ply is not a turning point** — system reports the error; nothing is changed
- **ply is 1** — system reports the error; the initial turning point cannot be removed

**Postconditions:**
- The two adjacent segments are merged into one segment
- The session remains open with unsaved changes
