# UC-16: View Segment

**Actor:** Author

**Preconditions:**
- A session is open for the game
- The segment exists

**Main Flow:**
1. Author selects a segment
2. System loads the working annotated PGN and working annotation JSON
3. System derives the segment's move range and extracts the move list for that span
4. System returns the segment's label, move list, annotation text, and diagram preview (if `show_diagram` is true)

**Alternate Flows:**
- **segment not found** — system reports the error; no segment is returned
- **`show_diagram` is false** — system returns the segment without a diagram preview

**Postconditions:**
- No state is changed
