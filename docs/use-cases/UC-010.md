# UC-10: Toggle Segment Diagram

**Actor:** Author

**Preconditions:**
- A session is open for the game
- The segment exists

**Main Flow:**
1. Author selects a segment
2. System flips the segment's `show_diagram` flag in the working annotation JSON
3. System returns the updated segment

**Alternate Flows:**
- **segment not found** — system reports the error; nothing is changed

**Postconditions:**
- The segment's diagram visibility setting is updated
- The session remains open with unsaved changes
