# UC-11: Save Game

**Actor:** Author

**Preconditions:**
- A session is open for the game

**Main Flow:**
1. Author requests save
2. System copies the working annotated PGN over the main annotated PGN
3. System copies the working annotation JSON over the main annotation JSON
4. System leaves the working files in place so the session remains open

**Alternate Flows:**
- **write fails** — system reports the error; the session remains open with the working files unchanged

**Postconditions:**
- The main files match the working files
- The session remains open
