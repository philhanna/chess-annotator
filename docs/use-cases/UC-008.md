# UC-8: Set Segment Label

**Actor:** Author

**Preconditions:**
- A session is open for the game
- The segment exists

**Main Flow:**
1. Author selects a segment
2. Author supplies a label
3. System updates the segment's label in the working annotation JSON
4. System returns the updated segment

**Alternate Flows:**
- **segment not found** — system reports the error; nothing is changed
- **label is blank** — system reports the error; nothing is changed

**Postconditions:**
- The segment has the new label
- The session remains open with unsaved changes
