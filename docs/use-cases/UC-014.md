# UC-14: Upload Original PGN to Lichess

**Actor:** Author

**Preconditions:**
- The game exists in the store

**Main Flow:**
1. Author requests Lichess upload for a game
2. System loads the game's `original.pgn`
3. System uploads the PGN to Lichess
4. System returns the analysis URL from Lichess

**Alternate Flows:**
- **game-id not found** — system reports the error; nothing is uploaded
- **upload fails** — system reports the error and returns no URL

**Postconditions:**
- The original PGN is available on Lichess at the returned URL
- No local game state is changed
